# Roadmap Archive

이 디렉토리는 현재 계획 후보로 읽히면 안 되는 과거 roadmap 자료를 보관한다.

## 구조

- `history/`: 긴 중기 계획 이력, 과거 실행 보드
- `reviews/`: 날짜가 붙은 health check, playtest review, post-review 문서

## 사용 규칙

- archive 문서는 historical context다.
- 현재 active slice와 다음 권장 후보는 `docs/roadmap/PROGRAM.md`가 우선한다.
- 현재 중기 보드는 `docs/roadmap/MIDTERM_TODO.md`가 우선한다.
- archive에서 아이디어를 되살릴 때는 먼저 현재 `PROGRAM.md`, domain spec, ExecPlan으로 승격한다.
